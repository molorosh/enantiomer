# enantiomer
A small web app using spaced repetition to help with memorising Japanese vocab.  

## Angular Stuff
This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 20.3.10.

the initial project kick-off command was:
```bash
ng new enantiomer --directory=. --standalone
```

I also chose Less for the styling (as I have never used that before!)

### Commands

To start a local development server, run:
```bash
ng serve
```

To generate a new component, run:
```bash
ng generate component component-name
```

To build the project run:
```bash
ng build
```

## Running unit tests

To execute unit tests with the [Karma](https://karma-runner.github.io) test runner, use the following command:

```bash
ng test
```
